    ##查询停车点详情（枫停）

    简要描述：

    • 查询枫停停车点详情

    请求URL

    • /v1/neighbourhood/{id}/parkPlaces

    请求方式

    • get

    参数

    参数名 必选 类型 说明
    id 是 int id

    返回示例

    返回参数说明

    返回参数包括 小区详情&车位详情，以下为变更&新增参数列表

    参数名 类型 说明
    parkPlaces.status int 车位状态，1可预订 2不可预订（显示为已被抢）
    longitude float 入口经度
    latitude float 入口纬度
    locationLongitude float 经度
    locationLatitude float 纬度
    processType int 车位流程类型 1普通 2万达（免扫码）
